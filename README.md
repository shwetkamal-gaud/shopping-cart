# Installation
```bash
git clone https://github.com/shwetkamal-gaud/shopping-cart.git
```
## Change the Directory
```bash
cd shopping-cart
```
## Install Dependencies 
```bash
npm install
```
## Run The Server
```bash
npm run dev
```
